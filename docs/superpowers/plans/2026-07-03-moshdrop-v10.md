## 任务

### T1 最小配置文件（④，先做——TTL/语言/开关都挂在它上面）
- `~/.moshdrop/config`，手写解析器（零新依赖，非 TOML 就叫 config）：
  ```
  # 全局
  ttl_days = 7          # 远端保质期，0=关闭清理
  intercept = on        # 拖拽拦截总开关
  lang = zh             # zh | en（用户可见文案）
  # 按主机覆盖：host.<别名>.键
  host.ccc.intercept = on
  host.ccc.remote_dir = .moshdrop
  ```
- 解析规则：k = v 行、# 注释、host.<name>.<key> 覆盖全局；未知键忽略（向后兼容）。
- intercept=off（全局或该 host）→ Uploader 直接 disabled（真·纯透传）。
- remote_dir 限定为 `$HOME` 下相对路径（拒绝 `/` 开头与 `..`，防逃逸）。

### T2 远端 TTL 自动清理（③）
- ensure() 的远端脚本用 `find -mtime +N -delete` 顺带清理正式文件（现有逻辑只清 .part 孤儿）；N 来自配置，0 则跳过。
- 只删普通文件不删目录；events.log 记一条 cleanup 事件（可选，简单起见 v1 不记）。

### T3 用户可见文案双语（①的一部分，英文优先）
- 消息目录：`msg(key)`，en 默认、zh 全量；来源优先级 config lang > $MOSHDROP_LANG > en。
- 覆盖：--help、doctor 全部标签、通知文案、错误分类文案。
- 给用户本机写好 `lang = zh` 的初始配置（体验不变）。
