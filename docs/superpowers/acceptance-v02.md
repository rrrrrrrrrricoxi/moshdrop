## 本轮现场抓获并根治的重大缺陷

1. **mosh 退出死锁**：输出停排空 → mosh `tcdrain` 卡死内核退出路径（SIGKILL 免疫）→ moshdrop 僵尸。根治：pty 永续排空 + Wait 限时收尾（5s KILL/3s 放弃）。
2. **保序回归（铁律①）**：同 chunk「粘贴+尾随字节」被重排（审查 3 维度独立确认）。根治：粘贴同步判定移入锁内按事件序执行。
3. **半截文件落正式名**：连接中途死亡 cat 误认 EOF 完成。根治：远端 `wc -c` 字节对账 + `ln` 原子占名（TOCTOU 免疫）+ 全失败路径清理。
4. **测试骚扰真实通知中心**：模拟故障用例触发真 osascript 通知。根治：`MOSHDROP_MUTE_NOTIFY` + TestMain 全局静音。

## 已知取舍（记录在案）

- 异步验真（500ms 硬上限防网络卷冻结）意味着"形似路径但非文件"的粘贴回放相对后续键入可能晚 ~ms 级——为"终端永不冻结"付出的固有代价。
