# moshdrop 市场备忘录（2026-07-04）

四路调研（痛点证据/人群规模/邻近参照/专职唱衰）结论存档，供未来决定投入度时翻阅。

## 判决

**需求真实但利基**：作为产品不成立；作为"真实小众刚需的最优解 + 自用刚需 + 开源名片"完全成立。
合理星数预期：几十~几百（参照赛道天花板 trzsz-ssh 2,652★）。
**退役条件**：Anthropic 官方修好 mosh 场景的图片粘贴（当前明确 not planned）。

## 支撑证据（关键数字）

### 人群在爆发
- Claude Code 年化 $2.5B+（2026/2），周活月翻倍，人均 20h/周
- "VPS 24/7 跑 Claude Code" 成熟内容品类（$6-12/月 Hetzner+tmux 标准配方）
- **mosh+tmux+Tailscale = 社区公认远程 agent 标准栈**（≥7 篇独立博客，含 Harper Reed 2026/1）
- 配套生态：Happy（远程 CC 手机端）一年 22,385★；Omnara YC S25；VibeTunnel 4,572★

### 痛点公开且官方摆烂
- anthropics/claude-code **#5277**（image paste in ssh）open 近一年，32👍/17评，评论区 "I NEED THIS!"，Anthropic 零回应
- OSC 协议提案 **#42712** 被官方 **3 分钟 closed not_planned**；#25672 同；#47519 关为 dup
- 2026/3-4 月集中冒出 **8 个社区工具**（cc-clip 120★/clipssh 48★/clipaste 30★…）
- openai/codex 同类痛点 11 个 open issue

### 天花板（诚实部分）
- 最热 issue 仅 32👍；8 个竞品全部两位数~百星；相关 Show HN 均 1-3 分
- 主流绕路：Cursor 本地 / 网页版自带上传 / VS Code Remote 2019 已解决拖传 / iTerm2 ⌥拖 scp 十年前就有
- 纯 SSH 场景已被 Claude Code 原生 OSC52 粘贴部分吃掉（部分终端组合可用）

### moshdrop 的缝（防御性）
- **8 个竞品全部依赖 SSH 隧道/转发 → 对 mosh 用户全部失效**
- Claude Code 原生粘贴走 OSC52，**mosh 不透传 OSC52**（mosh#637 长期 open）
- mosh 自 1.4.0（2022）无 release → 这条缝长期不会被上游填上
- ∴ 唯一卖点＝“mosh 会话原生可用、拖拽零配置”——README 定位已对齐
